# Available Themes

You can change the theme of your blog by updating the `theme` value in `config.json`.

## Built-in Themes

- `light` (Default) - A clean, high-contrast light theme (Shadcn Violet).
- `dark` - A modern dark theme (Shadcn Violet).
- `sepia` - A warm, paper-like theme, easy on the eyes.
- `nord` - A cool, arctic blue-grey theme based on the popular Nord color palette.
- `forest` - A nature-inspired theme with greens and off-whites.

## How to use

Edit `config.json`:

```json
{
    "theme": "nord",
    ...
}
```
