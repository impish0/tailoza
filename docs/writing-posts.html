Writing Posts

Tailoza supports full Obsidian-compatible markdown syntax. Write in Obsidian (or wherever), drop the file in posts/, and build.

Markdown Syntax

Text Formatting 
**bold**
*italic*
***bold and italic***
~~strikethrough~~
`inline code`
Headings 
# H1
## H2
### H3
#### H4
##### H5
###### H6
Lists 
Unordered:
- First item
- Second item
- Third item

Ordered:
1. Step one
2. Step two
3. Step three

Task lists:
- [x] Done thing
- [ ] Not done thing
Code Blocks

Fenced code blocks with syntax highlighting (powered by Prism.js):

```javascript
const greet = (name) => {
  console.log(`Hello, ${name}!`);
};
```
Blockquotes 
> "Some wise thing someone said once."

Nested blockquotes:

> Outer quote
>> Nested quote
Tables 
| Feature   | Supported |
|-----------|-----------|
| Markdown  | Yes       |
| RSS Feed  | Yes       |
| Search    | Yes       |
Images

Drop images in the images/ folder and reference them:

![Alt text](my-image.jpg)
Note: images can just be referenced by name — no need to include the path.
Links 
[Link text](https://example.com)
Horizontal Rules 
---
Download Links

Links to downloadable files (.pdf, .zip, .doc, etc.) automatically get a download arrow appended:

[Get the report](report.pdf)
Nested Lists

Indent with 2 spaces for nesting. Works with both ordered and unordered lists:

- Top level
  - Nested item
    - Even deeper
1. First
  1. Sub-item

Frontmatter

Every post starts with a frontmatter block — metadata between --- fences at the top of your markdown file.

Full Example 
---
title: My Cool Post
date: 2025-01-15
description: A short blurb about this post.
author: Your Name
categories: Technology, Tutorial
keywords: python, static site, blog
image: my-cover.jpg
toc: true
---

Your content starts here...
Required Fields
  • title — the post title
  • date — publish date in YYYY-MM-DD format
Optional Fields
  • description — used for SEO meta tags and the post list preview
  • author — overrides the default author from config
  • categories — comma-separated list (e.g. "Tech, Tutorial")
  • keywords — SEO keywords
  • image — cover image filename (from images/ folder)
  • toc — set to true to show a table of contents
Draft Posts

Prefix your filename with an underscore and it gets skipped during build. Simple as that.

posts/_my-draft-post.md   ← skipped
posts/my-published-post.md  ← built
